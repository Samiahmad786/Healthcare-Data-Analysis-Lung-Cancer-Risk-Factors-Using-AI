## üìÅ Dataset Description
The dataset contains anonymized healthcare-related information, including:

- **Demographics:** age, gender, education level  
- **Lifestyle factors:** smoking status, smoking years, exercise, alcohol consumption  
- **Environmental exposure:** air pollution index, passive smoking  
- **Clinical indicators:** BMI, CRP level, oxygen saturation, FEV1  
- **Target variable:** lung_cancer_risk
